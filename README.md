# Jogo Batalha Naval em Python

Projeto desenvolvido para o Tema 3 – Desenvolvendo a Lógica do Jogo.

## Regras do jogo
- O tabuleiro possui tamanho 5x5
- Existem 3 navios escondidos
- O jogador informa linha e coluna para atacar
- O jogo termina quando todos os navios são destruídos

## Como executar
1. Clone o repositório
2. Execute o arquivo `main.py`
3. Jogue pelo terminal
